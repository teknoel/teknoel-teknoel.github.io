Waiting for CubeCore
This page will automatically reload after CubeCore has rebooted and joined Wi-Fi, which typically takes less than 30 seconds. CubeCore will show a notification on the built-in display if there is a Wi-Fi network connectivity issue. Refer to the User Guide for troubleshooting.

CubeCore requires 2.4GHz secure Wi-Fi to be completely operational with all circuits functioning perfectly. Select to the same network that your other devices use to access the Internet. If your Wi-Fi network is not listed, try refreshing the list using . For networks that are not listed, you can enter the credentials manually.
