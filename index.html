This app features contextual help to explain every control and option. Toggle Help using the navigation bar above. A full User Guide is also available and can be accessed from the Main Menu. This guide provides information on using the built-in buttons to operate CubeCore without the app.

Use the slider to dim or turn off the 7-segment LED Display. When turned off, restores the display to the previous brightness.

The Clock has two display modes. Static mode always shows the Clock. Hide mode clears the display, and briefly illuminates the Clock through Tap Control. The mode buttons are only available when the brightness is turned up.

RD (Remote Display) will still update when the Display is turned off, but digit segments will appear in nifty shades of gray. Use RD to cycle through different Clock views: Time, Date, and Seconds. Simply click on RD to change the view. If CubeCore is showing a different function, clicking on RD will always return to the Clock.

The GPS subpanel shows a graphical representation of the number of satellites in view over time, with each segment's duration indicated by the value of Δ. The length of the vertical bars corresponds to the variation in the data collected within each period. The current count of satellites is displayed in large digits. Click on this value to display it on the LED Display. REC is the record number of tracked satellites since GPS was last started. Data is refreshed every five seconds. Reception blackouts are reported only if the period lasts more than a few minutes.

N/1000 = fraction of a second data. SYNC time stamp of last operation and countdown to next sync. TRIP is total internet travel time for sync transaction.

To manually sync, click on the countdown timer. This function is only available after five seconds from the last sync.

Select the reference source for synchronizing the clock: GPS (Global Positioning System) or NTP (Network Time Protocol). Use icon to reveal statistics about the active source.

GPS receives radio signals from earth-orbiting satellites. Time sync occurs every second by a hardware generated Pulse-Per-Second waveform with an accuracy of about 20 nanoseconds.

NTP receives the time from an internet server. Time sync occurs every 30 minutes with a typical accuracy of 20 milliseconds.

Sync Alert LED

Flash the built-in LED when GPS metadata is parsed. Data include the current date, time, and satellite statistics. Parsing occurs about every five seconds, separately from Pulse-Per-Second (PPS) signal.

Pulse Per Second

Pulse Per Second (PPS) blinks the built-in blue LED in sync with the time reference.
